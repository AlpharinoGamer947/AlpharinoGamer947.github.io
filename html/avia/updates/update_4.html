Today, Jelle (Dev) released a new updated. It is a UI overhaul of the Flight UI. We will show the new and old ui along with some new and QoL changes.



In-Game Settings

When trying to access settings, players had to leave the world and go to the main menu. But with this update, when we press esc, we can see that the settings button is there.

Old

New



Altimeter

Now left us check out the New Flight UI. First it is the Altimeter. The old altimeter was 4 small squares with numbers in them. this limited the max value to be represented to be 9999. But with the new UI, it looks similar to altimeters in airplanes. Actual mechanical altimeters are circular dials but here it is a rectangle with divides to show the height.

Old

New



Speedometer

Similarly to the altimeter, the speedometer is changed to depict the real world counterpart. The old one was the same as the altimeter. The new one is a dial with a needle pointing to the speed. The speed is displayed in 2 units, knots and Mach Speed. The speedometer shows both units. The number above the dial is in knots while the number under the dial is in mach speed.

Old

New



Throttle

The new throttle is like the speedometer, it is similar to real life throttle levers. From the horizontal throttle, it is now up-right.

Compass

Just like how we got the Settings Button in the pause menu, we got the new ui item, The Compass. With this compass, we can now know the direction we are facing. When we are not doing missions, the compass looks the same.

But when we are in a mission, the pointer to the destination airport will on the compass showing the direction of travel.

The UI for tutorial keybindings was straight forward. It was looking good for the old ui. But for the new UI, it looks better.
